# Sumit Beniwal Portfolio Website

A personal portfolio website built with Flask to showcase my experience, projects, technical skills, achievements, and contact details.

This portfolio highlights my work in backend development, computer vision, and AI-powered applications while keeping the interface clean, fast, and recruiter-friendly.

## Features

- Responsive personal portfolio built with Flask
- Updated experience section with both MultiTV internships
- Project showcase for InterViewPoint and Mini Metro Route Planner
- Technical skills section covering backend, AI/ML, databases, and tools
- Achievements section with competitive programming highlights
- Contact form integrated with WhatsApp
- Smooth scrolling, fade-in animations, and mobile navigation support

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd my-portfolio-website
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

### 3. Activate the virtual environment

On Windows PowerShell:

```powershell
.\.venv\Scripts\Activate
```

On macOS/Linux:

```bash
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the application

```bash
python run.py
```

Open the app in your browser:

[http://127.0.0.1:5000](http://127.0.0.1:5000)
